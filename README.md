[![demo](https://asciinema.org/a/330644.svg)](https://asciinema.org/a/330644?autoplay=1)

# Особенности

- Ограничил до 1 триллиона, при необходимости можно увеличить.
- В конце закомментирована возможность писать копейки числом.
